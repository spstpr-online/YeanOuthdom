# 🏫 CompSchool EduSys - ប្រព័ន្ធគ្រប់គ្រងសាលារៀន

**CompSchool EduSys** គឺជាកម្មវិធីគំរូ (Demo App) សម្រាប់គ្រប់គ្រងសាលារៀន ឬថ្នាក់រៀនកុំព្យូទ័រ (**Computer Class**) ដែលត្រូវបានអភិវឌ្ឍឡើងដោយប្រើប្រាស់ **React (JSX)** និង **Inline Styling**។ កម្មវិធីនេះមានប្រព័ន្ធបែងចែកតួនាទីអ្នកប្រើប្រាស់ច្បាស់លាស់ ចំណុចប្រទាក់ស្អាត និងងាយស្រួលយល់។

---

## 👤 តួនាទី និងមុខងារសំខាន់ៗ (User Roles & Features)

កម្មវិធីនេះត្រូវបានរចនាឡើងដោយបែងចែកជា ៣ តួនាទីចម្បង ទៅតាមទិដ្ឋភាពជាក់ស្តែងក្នុងសាលារៀន៖

### 1. អ្នកគ្រប់គ្រង (Admin)
* **Dashboard ទូទៅ:** មើលស្ថិតិសរុប (ចំនួនសិស្ស ការប្រឡង មធ្យមភាគពិន្ទុ និងក្រាហ្វបង្ហាញការចែកចាយកម្រិត Grade A-F)។
* **គ្រប់គ្រងសិស្ស:** ស្វែងរកឈ្មោះ ឬលេខកូដសិស្ស, បោះច្រោះ (Filter) តាមថ្នាក់, បន្ថែមសិស្សថ្មី, កែប្រែ ឬលុបទិន្នន័យសិស្ស។
* **ការប្រឡង:** មើលបញ្ជីការប្រឡងទាំងអស់ និងបង្កើតការប្រឡងថ្មី។
* **ពិន្ទុ:** ពិនិត្យមើលស្ថានភាពនៃការដាក់ពិន្ទុរបស់សិស្ស។
* **របាយការណ៍:** មើលលទ្ធផលសិក្សារបស់សិស្សប្រចាំថ្នាក់ និងចំណាត់ថ្នាក់ស្វ័យប្រវត្ត (🥇, 🥈, 🥉)។
* **ការកំណត់ (Settings):** ទំព័រសម្រាប់រៀបចំប្រព័ន្ធ (ព័ត៌មានសាលា, ឆ្នាំសិក្សា, គណនី, ការកំណត់ Grade Thresholds)។

### 2. គ្រូបង្រៀន (Teacher)
* **Dashboard ផ្ទាល់ខ្លួន:** មើលស្ថិតិរួម និងតារាងការចែកចាយពិន្ទុរបស់សិស្សក្នុងមុខវិជ្ជា Computer Science។
* **មើលបញ្ជីសិស្ស:** ពិនិត្យមើលព័ត៌មានសិស្ស និងលេខកូដសម្គាល់ខ្លួន។
* **គ្រប់គ្រងការប្រឡង:** បង្កើតការប្រឡងប្រចាំខែ ឬប្រចាំឆមាស/ឆ្នាំ ទៅតាមថ្នាក់នីមួយៗ។
* **ដាក់ពិន្ទុ (Scores Input):** ជ្រើសរើសការប្រឡង រួចបញ្ចូលពិន្ទុផ្ទាល់ដៃសម្រាប់សិស្សម្នាក់ៗ និងគណនាភាគរយ លទ្ធផលស្វ័យប្រវត្ត រួចចុចរក្សាទុក (Save)។
* **របាយការណ៍:** ពិនិត្យមើលលទ្ធផលរួមប្រចាំថ្នាក់ និងមានប៊ូតុងសម្រាប់ Export របាយការណ៍។

### 3. សិស្ស (Student)
* **Personal Dashboard:** ទំព័រស្វាគមន៍ឈ្មោះសិស្សផ្ទាល់ និងបង្ហាញស្ថិតិការប្រឡងរបស់ខ្លួន។
* **ការប្រឡងរបស់ខ្ញុំ:** ពិនិត្យមើលបញ្ជីឈ្មោះការប្រឡង ថ្ងៃខែ និងព័ត៌មានទាក់ទងនឹងការប្រឡង។
* **ពិន្ទុរបស់ខ្ញុំ:** មើលលទ្ធផលពិន្ទុដែលទទួលបាន, ភាគរយ, ចំណាត់ថ្នាក់ពិតប្រាកដនៅក្នុងថ្នាក់ (# Rank) និងកម្រិតនិទ្ទេស (A, B, C, D, F)។

## 🛠️ បច្ចេកវិទ្យាដែលប្រើប្រាស់ (Tech Stack)

* **Frontend Library:** React.js (Functional Components)
* **Hooks:** `useState`, `useEffect`, `useRef`
* **Styling:** Pure Inline Styles (ងាយស្រួលបត់បែន និងមិនបាច់ដើរហ្វាយ CSS ញញេរញញៃ)
* **Language:** ភាសាខ្មែរ និងអង់គ្លេស (Bilingual Interface)

## 🚀 របៀបដំណើរការកម្មវិធីនៅក្នុងគម្រោងរបស់អ្នក (How to Run)

ប្រសិនបើអ្នកចង់យកហ្វាយកូដ `school-management-app.jsx` នេះទៅ Run ក្នុងគម្រោង React របស់អ្នក (ឧទាហរណ៍៖ ប្រើជាមួយ Vite)៖

1. បង្កើតគម្រោង React ថ្មី ឬចូលទៅកាន់គម្រោងដែលមានស្រាប់។
2. បង្កើតហ្វាយមួយឈ្មោះថា `SchoolManagementApp.jsx` នៅក្នុងថត `src/`។
3. ចម្លង (Copy) កូដទាំងអស់ពីហ្វាយរបស់អ្នកដាក់ចូលក្នុងហ្វាយថ្មីនោះ។
4. ហៅទៅប្រើប្រាស់នៅក្នុង `App.jsx` ដើម៖
   ```jsx
   import SchoolManagementApp from './SchoolManagementApp';

   function App() {
     return <SchoolManagementApp />;
   }

   export default App;
